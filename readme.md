## Installation

```bash
git clone https://github.com/chipslays/imgfy.git imgfy
```

```bash
cd imgfy
```

```bash
composer install
```

```bash
npm install
```

```bash
npm run dev
```

For watch auto-changes `js` and `scss`:

```bash
npm run watch
```

For purge unused styles:

```bash
npm run prod
```
